# Security Policy

PersistGuard is a defensive, read-only auditing tool. It must never execute a discovered command, install a persistence item, or automatically remove one.

When reporting a security issue, do not include live credentials, private host inventories, or full scan reports in a public issue. Provide a minimal redacted reproduction and the affected version.

The HTML report embeds local scan evidence. Treat generated reports as sensitive host inventory and share them only with intended reviewers.
